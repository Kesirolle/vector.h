Detailed Description

Abstract base class that inherits from Referent, and adds the ability for instances to be referenced by a WeakReferentPtr.

Any WeakReferentPtr will Acquire only NULL after the last ReferentPtr (a.k.a. "strong reference") to the instance goes away.

All Referent usage guidelines (regarding copying, private/protected destructors, etc.) also apply to WeakReferent.
